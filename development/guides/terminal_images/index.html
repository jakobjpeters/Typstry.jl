Terminal Images

Some terminals implement protocols that enable direct image rendering. If the current terminal supports a given protocol, simply load the corresponding package in the REPL to automatically render values of type AbstractTypst and TypstString.

ITerm2Images.jl

  • [✓] wezterm
  • [?] iTerm2
    • Very likely to work, but not yet confirmed.
  • [❌] rio
    • Rio claims to support the iTerm2 image protocol, but fails to render anything.
  • others?
