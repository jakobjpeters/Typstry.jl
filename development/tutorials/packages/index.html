MakieTeX.jl

Note

This package re-exports @typst_str and TypstString.

julia> using CairoMakie, MakieTeX
julia> f = Figure(; size = (100, 100));
julia> LTeX(f[1, 1], TypstDocument(typst"$1 / x$"); scale = 5);
julia> save("makie_tex.svg", f);
